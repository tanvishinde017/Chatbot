# 🤖 Advanced AI Chatbot (Python)

A machine learning based chatbot built using Python, Flask, and Scikit-learn .

## 🚀 Features
- Flask backend
- NLP intent classification
- Machine Learning (Naive Bayes)
- Web-based chat UI

## 🛠 Tech Stack
- Python
- Flask
- Scikit-learn
- HTML, CSS, JavaScript

## ▶️ Run Locally
```bash
pip install -r requirements.txt
python app.py
